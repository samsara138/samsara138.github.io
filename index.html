Projects

PopcornShell

A simple python reverse shell project, where sockets are used to establish a connection between the client and the server. And the server can send commands for the client to execute using subprocesses.

GalMaker

A Django web app project. Gal game is a game genre that's know for it's a narrative-driven game play, where the player make choices during the dialog that have consequences in the story. This project includes a builder to make gal game and a player to play those builds.

MacroMaker

A python based macro maker and player, simulate authentic mouse and keyboard input to automate tasks. The macro is constructed in a csv file, using assembly like instructions for command and logical behavior such as loops and variable assignment.
